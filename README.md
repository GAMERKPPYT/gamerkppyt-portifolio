# gamerkppyt-portfolio

Portfólio pessoal desenvolvido do zero com HTML, CSS e JavaScript puro — sem frameworks, sem dependências.

🌐 **Live:** [gamerkppyt.github.io/gamerkppyt-portfolio](https://gamerkppyt.github.io/gamerkppyt-portfolio)

---

## Sobre

Site de portfólio pessoal que apresenta minha stack, projetos e formas de contato. Desenvolvido com foco em performance, animações fluidas e design responsivo.

---

## Stack

| Tecnologia | Uso |
|---|---|
| HTML5 | Estrutura semântica |
| CSS3 | Animações, layout (Grid/Flexbox), variáveis |
| JavaScript (ES6+) | Interatividade, canvas, efeitos de digitação |

---

## Funcionalidades

- Cursor customizado com efeito de hover
- Background animado via Canvas API
- Efeito de digitação no bloco de código
- Filtro de tech cards por categoria (Backend / Frontend)
- Barras de idioma animadas com IntersectionObserver
- Scroll reveal nas seções
- Design totalmente responsivo (320px → 4K)
